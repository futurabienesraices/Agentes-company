# SKILL: Producción de Contenido con Assets Visuales
> **ID:** produccion_contenido | **Versión:** 1.0 | **Creado:** 2026-04-19

---

## Propósito

Convertir los datos de una propiedad o servicio, junto con los assets visuales disponibles en Google Drive, en contenido listo para publicar en redes sociales.

Output siempre accionable: guión para grabar, copy para publicar, brief para diseñar. Nunca ideas genéricas.

---

## Entradas Esperadas

| Campo | Requerido | Descripción |
|---|---|---|
| Ficha de propiedad o servicio | Sí | Datos completos del `BR-XXX.md` o briefing del servicio |
| Formato objetivo | Sí | Reel / Carrusel / Flyer / Anuncio pagado / Post de texto |
| Assets disponibles | No | Link de Drive o indicación de fotos/videos disponibles |
| Público objetivo | Sí | inversionistas / familias / emprendedores / hogar |
| Plataforma | Sí | Instagram / Facebook / WhatsApp Status / Story |

---

## Proceso Paso a Paso

### 1. Verificar assets

```
¿Hay link_carpeta_drive en la ficha?
├── SÍ → Indicar en el output qué fotos/videos usar en cada parte
└── NO → Producir el contenido igual + incluir sección "FOTOS NECESARIAS"
          indicando exactamente qué tomas se deben conseguir
```

### 2. Definir el ángulo

Elegir UN ángulo principal por pieza. No mezclar:

| Público | Ángulo principal |
|---|---|
| Inversionistas | Plusvalía, rentabilidad, crecimiento de la zona |
| Familias | Espacio, seguridad, comunidad, sueño propio |
| Emprendedores | Oportunidad, visibilidad, crecimiento del negocio |
| Clientes Cleaning | Salud, higiene profunda, evidencia visual |

### 3. Seleccionar estructura según formato

Ver especificaciones completas en `context/produccion_contenido.md`.

### 4. Producir el output completo

Entregar todo en un solo bloque: copy listo, indicaciones visuales, hashtags, CTA.

## Reglas de Producción

1. **Assets primero, copy después.** Verificar qué hay disponible antes de indicar formatos.
2. **Un ángulo por pieza.** No mezclar inversión + familia en el mismo contenido.
3. **Rotación de formatos.** Por propiedad: Reel → Carrusel → Flyer → Anuncio. No repetir el mismo formato consecutivo.
4. **Siempre incluir CTA.** Ninguna pieza termina sin un siguiente paso claro.
5. **Si no hay fotos,** producir igual y agregar la sección `FOTOS NECESARIAS` con instrucciones exactas de qué tomar.
6. **Copys cortos para móvil.** El 80% del tráfico lee desde celular. Párrafos de 1–2 líneas máximo.
